How to push to gitHub ?

When you are at a new machine, you want to configure yourself as the author of the projects. Do this by running the following commands:

git config --global user.name "your-github-name"

git config --global user.email your-github-email

When you are ready to push your project to Github, press Alt+F12 in IntelliJ to open the terminal window. You can see that we are already in the current project folder. For instance, if my project was named SpringBoot_102, you would see:

C:\Users\GBTC406001ur\IdeaProjects\SpringBoot_102>

Once we confirm that we are in the right directory, type the following to link your Github repository with the local project.This is also the link you would be submitting to Canvas :

git init

git remote add origin your-github-repository-link

To add and commit your work, and push it to Github, type the following commands:

git add .

git commit -m "whatever-commit-message"

git push -u origin master

Let's say you want to work on this project on your home computer. You then have to "pull" what you "pushed" earlier. Run this command, then click anywhere on your IntelliJ workspace to load the changes:

git pull
