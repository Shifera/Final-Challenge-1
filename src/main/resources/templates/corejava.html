What is Core Java?



"Core Java" is Sun's term, refers to Java SE, the standard edition and a set of related technologies, like the Java VM. This is mostly to differentiate from, say, Java ME or Java EE. Also note that they're talking about a set of libraries rather than the programming language. Java is one of the most popular programming languages in use today. Java aims to allow the programmer to write once, run anywhere. The Core Java technologies and application programming interfaces (APIs) are the foundation of the Java Platform, Standard Edition (Java SE). They are used in all classes of Java programming, from desktop applications to Java EE applications.In theory you should be able to run the same Java program on a linux machine as a Windows machine.

Java was developed in the early 1990's at Sun Microsystems by James Gosling. Sun has since been acquired by Oracle Corporation,so they now maintain Java. Java is one of the most popular programming languages in use today.
